# Profile/Portfolio project made using ReactJS/Tailwind/Vite
# Design inspired by https://www.figma.com/file/FY5ahab5jxG8RoecLP45eZ/Portfolio-(Community)?type=design&node-id=0-1&t=a1CItNCtUsipu0Gx-0

## Deployment:

```
> npm install
```

## Run the project:

### For development:
```
> npm run dev
```
or
```
> vite
```

### Build:
```
> npm run build
```
or
```
> vite build
```

### Preview
```
> npm preview
```
or
```
> vite preview
```

Good luck, have fun!
